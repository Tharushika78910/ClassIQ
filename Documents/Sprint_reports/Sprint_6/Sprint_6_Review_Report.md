# Sprint 6 Review Report

## Sprint Goal

The goal of Sprint 6 was to add multilingual support to the database and connect it with the system. This included updating the database structure, modifying backend queries, and making sure the system works properly in different languages.

---

## Completed User Stories / Tasks

* Prepared Sprint 6 planning
* Modified the database to support multiple languages
* Created translation tables and inserted multilingual data (English, Sinhala, Arabic)
* Updated backend (DAO classes) to retrieve data based on selected language
* Connected UI with backend localization
* Localized UI pages:

    * Login Page (Roshini)
    * Teacher Student Details Page (Kumudu)
    * Student Report Card Page (Chathu)
    * Teacher Marksheet Page (Dilmi)
* Completed and verified localization Excel sheet
* Ran SonarQube analysis (mainly done by Poornima and Kumudu)
* Created sequence diagram (Dilmi)
* Updated README and Scrum documents (Poornima)
