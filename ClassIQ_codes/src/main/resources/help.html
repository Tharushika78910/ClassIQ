ClassIQ Help

Getting Started

To use ClassIQ, start by logging in with your teacher or student account. Teachers can manage student grades and assignments, while students can view their grades and report cards.

Features

  • Login/Logout functionality
  • Student grade management
  • Weighted grade calculations and report card generation
  • Search functionality for students and assignments
  • Automated system with data validation

Navigation Tips

Use the main buttons to access dashboards and features. Click on "Privacy Policy" in the footer to view privacy information.

Support

If you encounter any issues, contact the project development team via email or report issues in the system documentation.
